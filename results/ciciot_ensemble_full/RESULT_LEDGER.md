## Claim Control

- The ensemble is valid only as an internally tuned model because alpha is chosen on a split of the training file.
- The test file is not used for alpha selection.
- If the composite objective sacrifices too much Macro-F1, it should be written as a recall-sensitive operating point rather than the default model.
